## Requisitos de Aprendizagem

- **01** (Lembrar/Entender): O aluno deve ser capaz de identificar as tabelas e colunas necessárias para monitorar os indicadores da cidade (energia, tráfego e segurança) através de consultas simples.

- **02** (Aplicar): O aluno deve executar filtros específicos utilizando a cláusula WHERE para localizar quais distritos ou setores da cidade estão operando em estado crítico.

- **03** (Analisar): O aluno deve utilizar funções de agregação e ordenação (como AVG, SUM e ORDER BY) para analisar o consumo de recursos por zona e priorizar as intervenções mais urgentes.

- **04** (Criar/Sintetizar): O aluno deve construir comandos de manipulação de dados (UPDATE) para modificar o estado dos sistemas da cidade, sintetizando soluções que equilibrem o orçamento e a eficiência dos serviços.
