# Projeto de Objetos de Aprendizagem
**Tema:** Banco de dados

---

### Sobre o neonCity: Administrando a Cidade
O neonCity é um Objeto de Aprendizagem gamificado, desenvolvido em React, que coloca o usuário no controle total de uma metrópole futurista. O objetivo é ensinar os fundamentos de Banco de Dados de forma prática, visual e imersiva.

### A Premissa
Em um futuro onde a estabilidade urbana depende de sistemas de dados complexos, você assume o cargo de Administrador do Sistema Central. A cidade está sofrendo com instabilidades: apagões, falhas de segurança e colapsos logísticos são representados visualmente por áreas em vermelho neon no seu mapa interativo.

### Como Funciona
Para salvar a cidade, o jogador não utiliza botões comuns, mas sim o terminal de comando SQL.

- **Diagnóstico**: Utilize comandos SELECT e cláusulas WHERE para identificar quais distritos estão em estado crítico.

- **Ação**: Execute comandos UPDATE para restaurar a carga dos reatores ou redirecionar o tráfego.

- **Resultado**: Através do processamento em tempo real, cada consulta correta reflete instantaneamente na interface. Ao estabilizar um setor, a área no mapa pulsa de vermelho para um verde neon vibrante, simbolizando o sucesso da operação.

### Objetivos Educacionais
- O projeto utiliza a Taxonomia de Bloom e a metodologia ágil AM-OER para garantir que o aprendizado seja progressivo:

- Exploração: Familiarização com tabelas e colunas.

- Operação: Aplicação de filtros e ordenação de dados.

- Gestão: Modificação de estados de banco de dados para resolução de problemas complexos.

### Requisitos de Aprendizagem

- **01** (Lembrar/Entender): O aluno deve ser capaz de identificar as tabelas e colunas necessárias para monitorar os indicadores da cidade (energia, tráfego e segurança) através de consultas simples.

- **02** (Aplicar): O aluno deve executar filtros específicos utilizando a cláusula WHERE para localizar quais distritos ou setores da cidade estão operando em estado crítico.

- **03** (Analisar): O aluno deve utilizar funções de agregação e ordenação (como AVG, SUM e ORDER BY) para analisar o consumo de recursos por zona e priorizar as intervenções mais urgentes.

- **04** (Criar/Sintetizar): O aluno deve construir comandos de manipulação de dados (UPDATE) para modificar o estado dos sistemas da cidade, sintetizando soluções que equilibrem o orçamento e a eficiência dos serviços.
